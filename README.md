📊 Inventory Management Dashboard (SQL + Power BI)

This project focuses on analyzing inventory data to monitor stock levels, track product performance, and support data-driven decision-making. The workflow includes data extraction and transformation using SQL followed by interactive dashboard creation in Power BI.

🔹 Key Features
KPI tracking for Total Revenue, Orders, and Product Count
Analysis of Top 5 products by remaining stock
Identification of low-stock items for better inventory planning
Interactive filters for product-level and time-based insights
🔹 Tools & Technologies
SQL – Data cleaning, joins, and aggregation
Power BI – Data visualization and dashboard development
🔹 Insights Delivered
Highlighted products with declining inventory levels
Enabled better stock monitoring and replenishment decisions
Provided a clear overview of inventory performance
